## Classic RL algorithms

---
## Overview
In this repo, the term '**Classic RL**' is used for these basic RL algorithms.

* Model-free Monte Carlo
* SARSA
* Q-learning
* Dyna-Q


---
## Environments
**Classic RL** algorithms are applicable to gridworld environments.
1. BlackJack
2. Taxi
3. CliffWalking
4. FrozenLake
